version 1.0, 16 July 2002

-------------------------------------------------------

Installation instructions:

This package includes a framework (DisclosableView.framework) and an Interface Builder palette (DisclosableView.palette).  Copy the framework to /Library/Frameworks on your startup disk.  If you put the palette in any of these locations:
            ~/Library/Palettes
            ~/Developer/Palettes
            /Library/Developer/Palettes
            /Developer/Palettes
            /Network/Developer/Palettes
then Interface Builder will find it when it starts up.  Or, you can put the palette somewhere else, and tell IB where it is using the "Palettes" tab in the preferences.
